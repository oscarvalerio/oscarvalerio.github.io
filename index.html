About Me

Valerio Gomez Oscar is the Software Test Engineer behind VAGO Testing, Working as Software Tester, I started back in 2005 as manual tester, after that, it has been a long but funny road by different areas of the Software Quality Assurance area. My experience is mainly with Automation Test Tools like Selenium, UFT, and Appium, but I also have experience with other areas of testing like Performance Testing (LoadRunner), API testing (PostMan, SOAP UI), Security Testing, and Test Management. The goal for this site is to create a place to explore automation for spanish speakers mainly but having a centralized place to share knowledge.
